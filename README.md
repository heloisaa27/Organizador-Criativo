Organizador Criativo
- Um sistema para organização de projetos criativos, com foco em personagens, relações e estrutura narrativa.

Tecnologias

- Frontend: React + Vite

- Backend: Node.js + Express

- Banco de dados: SQLite

1. Clone o projeto:

git clone https://github.com/heloisaa27/Organizador-Criativo.git

2. Entrar na pasta(caso ainda não esteja):
- cd Organizador-Criativo

3. Instale as dependências:

- npm install

- npm run install:all

4. Como rodar o projeto?
- npm run dev

Isso irá iniciar:

- Frontend(React)

- Backend(API)


CYPRESS

- cd frontend

- npx cypress open
